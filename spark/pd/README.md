# Spark Firmware: Pure Data (Pd) & PlugData Runtime

This folder will contain projects and tools for running Pure Data (Pd) and PlugData patches on the Spark module, via Daisy Seed.

- **Purpose:**
  - Develop and deploy audio patches using Pure Data (Pd) or PlugData, targeting Daisy hardware with Oopsy.
  - PlugData is a modern, cross-platform fork of Pure Data with a streamlined interface and VST/AU support.
  - Oopsy is the toolchain that converts Pd/PlugData patches into firmware for Daisy Seed-based devices like Spark.

- **Documentation:**
  - [Daisy Audio Software Overview](https://daisy.audio/software/)
  - [Pure Data (Pd)](https://puredata.info/)
  - [PlugData](https://plugdata.org/)
  - [Oopsy for Daisy](https://github.com/electro-smith/oopsy)
