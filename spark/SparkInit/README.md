## Controls

In any mode, click the encoder and then turn to select different mode. Click again to confirm.

| Mode | Color | Turn Encoder | Turn Knob 1 | Turn Knob 2 | Tap Switch 1 | Tap Switch 2 | Hold Switch 1 | Hold Switch 2 |

|  Filter |  Yellow |  |  |  |  |  |  |  |
|  Envelope |  Orange |  |  |  |  |  |  |  |
