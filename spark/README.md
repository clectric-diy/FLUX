# Spark Firmware Runtimes

This directory contains firmware and runtime projects for the Spark module, supporting multiple development environments and languages for the Daisy Seed platform.


## Subfolders

- **cpp/** — C++ firmware (libDaisy, DaisySP)
- **ino/** — Arduino sketches (DaisyDuino)
- **max/** — Max/MSP externals (Oopsy, Gen~)
- **pd/** — Pure Data & PlugData patches (Oopsy)
- **rs/** — Rust firmware (daisy-rs)

Each runtime supports different workflows and skill levels—see the README in each subfolder for details and documentation links.
