# Spark Firmware: Rust Runtime

This folder will contain Rust-based firmware and tools for the Spark module.

## Purpose

- Projects and code for programming Spark using the Rust language and Daisy Seed support crates.
- For advanced users and experimental firmware development.
- Community-driven and evolving support for Daisy Seed in Rust.

## Main Crate & Tools

- [daisy-rs (Rust Daisy)](https://github.com/ryanisaacg/daisy) — Main Rust crate for Daisy Seed
- [daisy-rs on crates.io](https://crates.io/crates/daisy)
- [daisy-rs Examples](https://github.com/ryanisaacg/daisy/tree/main/examples)
- [cargo-embed](https://github.com/probe-rs/cargo-embed) — Flash Rust firmware to hardware
- [PlatformIO](https://platformio.org/)
- [Daisy Web Programmer](https://programmer.electro-smith.com/) — Flash firmware via browser (if compatible)

## Documentation & Resources

- [Daisy Audio Software Overview](https://daisy.audio/software/)
