# Spark Firmware: Arduino (INO) Runtime

This folder will contain Arduino-compatible firmware for the Spark module.

## Purpose

- Projects and code for programming Spark using the Arduino IDE and .ino sketches.
- Ideal for rapid prototyping and users familiar with Arduino.
- Supports both Arduino IDE and PlatformIO.

## Main Library & Tools

- [DaisyDuino](https://github.com/electro-smith/DaisyDuino) — Arduino library for Daisy Seed
- [Arduino Library Manager](https://www.arduino.cc/en/guide/libraries) — Install DaisyDuino easily
- [DaisyExamples (Arduino)](https://github.com/electro-smith/DaisyExamples/tree/master/seed/Arduino) — Example sketches
- [Daisy Web Programmer](https://programmer.electro-smith.com/) — Flash firmware via browser
