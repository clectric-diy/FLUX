# spark Ocsillators

## Author
Charles H. Leggett (chleggett)

## Description
A multi-mode oscillator for the clectric Spark module.

[Source Code](https://github.com/clectric-diy/FLUX/tree/main/spark/cpp/oscillators)

## Controls

In any collection, click the encoder and then turn to select different collection. Click again to confirm.

| Collection       | Color  | Turn Encoder    | Turn Knob 1 | Turn Knob 2 | Tap Switch 1 | Tap Switch 2 | Hold Switch 1 | Hold Switch 2 |
|------------|--------|-----------------|-------------|-------------|--------------|--------------|---------------|---------------|
| Waveforms | White  | Change Waveform |             |             |              |              |               |               |
| MacroBankA     | Yellow |                 |             |             |              |              |               |               |
| MacroBankB   | Orange |                 |             |             |              |              |               |               |
| TBD    | Green  |                 |             |             |              |              |               |               |
| TBD        | Blue   |                 |             |             |              |              |               |               |
| TBD        | Purple |                 |             |             |              |              |               |               |
| TBD        | Red    |                 |             |             |              |              |               |               |
